   link: https://yadi.sk/d/gSeGdF6coO_brQ

2. Raw part of RuDReC. \
   link: https://yadi.sk/d/kCsAhkoLZUuTrQ

   link: https://yadi.sk/d/-PTn0xhk1PqvgQ
